Big Data is Our A to Z

분당 페이스북에는 2000만번 이상의 자료 공유가, 트위터에는 30만번 이상의 트윗이, 인스타그램에는 22만장 이상의 사진이, 유튜브에는 72시간짜리 영상이, 구글에는 하루 24테라바이트의 데이터가 업로드되고 있습니다. 더욱 가속화되고 있는 데이터의 증가 추이, 규모를 가늠할 수조차 없는 수많은 정보의 폭발, 빅데이터는 미래 경쟁력의 핵심 자원으로 떠오르고 있습니다.

이렇게 많은 데이터로 어떤 일을 할 수 있을까요? 어디에 어떤 방향으로 접근해야 유의미한 결과를 도출할 수 있을까요? 빅데이터란 정말 '크고 많은' 데이터만을 지칭하는 말이며, 큰 데이터가 아니면 의미가 없는 것일까요?

BOAZ는 이러한 질문들을 끊임없이 던지면서 토론과 실습을 통해 답을 찾아가려고 노력합니다. BOAZ는 데이터로부터 긍정적인 가치를 이끌어 빅데이터 시대의 인재로 거듭나고자 하는 대학생들이 모여 있는 동아리입니다. BOAZ는 다양한 이공계/인문계/예술계열 학생들로 구성된 가치관과 시각의 집합소입니다.

Our Services

BOAZ는 데이터를 수집하여 분석하고 머신러닝 및 딥러닝 모델링을 실시하는 데이터 분석 세션,
정보 전달의 효율성을 극대화하기 위해 데이터를 통해 인사이트를 도출하는 데이터 시각화 세션,
그리고 데이터 적재부터 전처리, 모델링 결과의 활용까지 데이터 처리의 전반적인 흐름을 정의하는 데이터 엔지니어링 세션으로 나뉩니다.
세 분야는 서로 다른 커리큘럼을 바탕으로 진행되며, 각 달에 한 번씩 공동세션을 통해 데이터 전반에 대한 생각을 나누고 친목을 도모합니다.

분석 세션

데이터 수집, 전처리, 그리고 ML/DL을 통한 모델링까지 전반적인 데이터 분석을 아우르는 세션입니다.
수집한 데이터를 바탕으로 자신의 목적에 알맞게 데이터를 변형할 수 있습니다
데이터를 여러가지 모델에 적용하여 데이터 전반에서 알 수 없던 결과를 도출하고 이를 활용하는 능력을 기릅니다.

방학 기간

머신러닝 및 딥러닝 기초 이론을 배우고, 실습을 진행.

기초 분석 방법론

Python을 이용하여 효과적인 데이터 전처리를 위해 데이터 구조에 대해 학습하고, 사용할 데이터를 가져와 재구성하는 방법을 배웁니다.
데이터 처리 및 활용에 효과적인 Numpy, Pandas, Seaborn 등의 Python 라이브러리에 대해 배우고 실습해보면서 데이터 처리를 학습합니다.

머신러닝(ML) 기초

컴퓨터가 스스로 데이터를 분석해 의미있는 결과를 도출하는 머신 러닝에 대해 학습하고 실제로 적용합니다.
Regression, Classification과 Regularization, Feature Analysis 등의 다양한 개념들을 공부합니다.
배운 내용을 바탕으로 Kaggle에서 진행된 Competition을 주제로 프로젝트를 진행합니다.

딥러닝(DL) 기초

Multi-Layer Perceptron(MLP), Backpropagation, Gradient Descent, Convolutional Neural Network(CNN) 등 딥러닝의 기본 이론과 기법들을 공부합니다.
이를 바탕으로 Pytorch, Keras, Tensorflow 등의 딥러닝 프레임워크를 이용해 직접 구현해보는 시간을 갖습니다.

학기 기간

방학 중 배운 머신러닝 알고리즘을 직접 실습해보고 딥러닝 논문을 읽고 구현.

딥러닝(DL) 심화

방학동안 배운 딥러닝 이론을 바탕으로 NLP, Vision 등 관심 Task를 정해 집중적으로 공부합니다.
각 분야에서 대표적인 논문들을 중심으로 공부하고, 이를 바탕으로 자신이 원하는 주제를 정하여 Tensorflow, Pytorch 등 딥러닝 프레임워크를 활용해 구현합니다.

자연어 처리

자연어처리(Natural Language Processing, NLP)는 자연언어의 의미를 분석하고 해석하는 분야로, 언어의 의미/문법적 정보를 설명하고 문맥적 의미를 더 잘 파악하는 방향으로 발전했습니다.
단어의 출현 빈도와 관계성을 파악하는 단순 텍스트 마이닝의 차원을 넘어 기계번역, 감성분석, 대화생성, 질의응답까지 응용됩니다.
말뭉치와 문법정보를 함축하기 위한 임베딩 기초 모델 Word2vec, Seq2seq 모델을 시작으로 Attention, Transformer, Ber 등을 공통으로 공부한 후 최신 NLP 모델의 논문을 공부합니다.

컴퓨터 비전

컴퓨터 비전은 딥러닝이 성공적으로 적용된 분야 중 하나이며, 대표적인 분야는 이미지 분류, 물체 인식, 이미지 생성이 있습니다.
방학동안 배운 CNN을 바탕으로 VGG, Resnet, Inception, DenseNet을 공통으로 공부하면서 컴퓨터 비전의 기초를 다집니다.
이후에 세부 Task에 대해 집중적으로 학습하고 발표를 진행합니다. RCNN과 YOLO, VAE, GAN 등을 이용한 모델을 통해 각 Task를 공부합니다.

시각화 세션

통계분석으로는 알 수 없는 데이터의 이야기를 끌어내 숨겨진 인사이트를 도출하는 것을 목표로 하는 세션입니다.
데이터를 통해 원하는 인사이트를 뽑아내고 이를 남들에게 제공 및 소통하는 방법을 배웁니다.
본인만의 데이터에 관한 철학을 얻고, 나아가 개성을 살린 데이터 스토리텔링 능력을 갖춘 인재로 성장합니다.

방학 기간

Pytho을 이용한 프로그래밍 학습, Tableau를 이용한 데이터 시각화 학습, 케이스 스터디 진행.

프로그래밍 학습

Python을 통해 데이터 수집 및 데이터 전처리를 위한 기초 프로그래밍을 학습합니다.
데이터를 수집하기 위한 웹크롤링과 데이터의 특성을 전반적으로 파악하기 위한 전처리 및 EDA 방법들을 실습합니다.
이를 통해 최적의 데이터를 준비하는 방법을 배웁니다.

방학 스터디

기존 기수와 신입 기수가 함께 Tableau를 활용한 대시보드 설계를 구성하는 방법을 실습합니다.
방학 스터디를 통해 Drag&Drop만으로 유용하게 시각화가 가능한 툴 Tableau의 다양한 기능을 사용하여 데이터의 숨겨진 중요한 의미들을 빠르고 효과적으로 시각화하는 방법을 배웁니다.

케이스 스터디

매주 과제를 통해 중요한 정보를 전달하기 위한 차트를 직접 구상하고 발표합니다.
다양한 관점을 가진 회원들과 자유롭게 의견을 공유하고 시각화와 관련해 토론하는 시간을 가지면서 문제를 다각도로 바라보는 힘을 기릅니다.

학기 기간

데이터 시각화 주제를 선정하여 데이터 분석부터 시각화까지 자율적인 프로젝트를 진행.

데이터 분석 심화(Python)

방학동안 배운 데이터 전처리 기법을 바탕으로 더 심층적인 기법들을 학습합니다.
다양한 데이터 유형별 전처리 기법을 실습하는 것부터, 간단한 머신러닝 및 자연어 처리를 비롯해 자신의 관심사에 맞는 데이터 분석 기법을 공부합니다.

팀별 프로젝트(Tableau)

방학 세션 및 다양한 활동을 기반으로 자유롭게 주제를 선정한 후, Base 팀별 프로젝트를 진행합니다.
데이터 수집부터 전처리, 분석 및 시각화까지 모든 단계를 수행해 그 결고물을 충실히 보여줄 수 있는 컨텐츠까지 제작합니다.
이후 팀별 프로젝트의 결과물은 컨퍼런스에서 발표 및 전시합니다.

엔지니어링 세션

하나의 저장소에 저장 및 분석이 어려운 빅데이터를 데이터베이스와 Hadoop을 통해 효율적 관리와 운영을 추구하는 세션입니다.
빅데이터에 관한 효율적인 관리 능력을 키울 수 있습니다.
나아가 파이프라인 구축을 통해 데이터 수집부터 ML/DL 분석까지의 과정에 대해 Real-Time 수준의 데이터 운용 실력을 키웁니다.

방학 기간

리눅스와 가상화, DBMS, SQL 등 데이터 파이프라인 구축을 위한 기초를 다짐.

리눅스 기초

먼저 원활한 데이터 파이프라인 처리를 위해 컴퓨터 OS 커널의 일종 혹은 리눅스 커널을 사용하는 운영체제인 리눅스의 기초에 대해 배웁니다.
Shell Script 프로그래밍을 통한 실습으로 리눅스 명령어를 해석하는 기능의 Bash Shell부터 리눅스 파일 시스템의 구조 및 특징, 프로세스 및 패키지 관리 등 전반적인 리눅스 기능에 대해 학습합니다.

네트워크 기초

우리가 사용하는 컴퓨터에서 빅데이터가 저장되어 있는 데이터베이스에 접근하고 데이터를 처리하기 위해서는 컴퓨터를 연결해 데이터 통신을 제공하는 네트워크가 필요합니다.
네트워크 기초를 학습함으로써 이후 데이터 엔지니어링 분야에서 많이 활용하는 가상화 기술에 대해 학습할 때, 높은 진입장벽을 낮춰줍니다.

데이터베이스 기초

RDBMS, ER-Model, 정규화 등 데이터베이스의 기본이 되는 이론을 공부합니다.
이를 바탕으로 SQL을 이용해 직접 데이터베이스에서 데이터를 구축하고 처리, 분석해봅니다.
확장성, 유연성, 가용성 등 빅데이터 시대의 요구사항을 충족하지 못한 SQL의 한계로 등장한 NoSQL의 기초를 배웁니다.

가상화 기술

대표적인 가상화 기술인 Docker에 대해 학습합니다.
컨테이너, 가상 머신, 클라우드 환경 등의 가상화 기술에 대한 기초적인 이론에 대해 배웁니다.
이후, 리눅스 커널 기반으로 구성된 Docker의 컨테이너 기술을 이용해 실습을 진행합니다.

학기 기간

분산 처리, 실시간 처리 오픈 소스를 직접 다뤄보고 데이터 운영 및 관리를 진행.

범용 빅데이터 저장소 구축

방학동안 배운 기초 이론을 바탕으로 빅데이터를 위한 범용 저장소 및 분석 플랫폼을 제공해주는 하둡(Hadoop)을 공부합니다.
하둡의 Map-Reduce, Yarm, HDFS 등의 이론을 통해 분산 컴퓨팅에 대한 기초를 학습합니다.

인메모리(In-Memory) 분산 데이터 처리

스파크(Spark)는 메모리 상 동작하는 분산 데이터 분석 시스템으로 하둡의 Map-Reduce 모델에 대화형 명령어 쿼리, 스트리밍, ML이 가능하도록 확장한 프레임워크입니다.
Spark의 Application 실행 구조같은 특징과 스파크의 RDD 데이터 타입에 대한 이론 등을 학습합니다.
스파크 인터페이스인 Spark Shell과 Zeppelin Notebook을 통해 데이터프레임을 생성하는 등의 실습을 진행합니다.

실시간 메시지 처리

통합된 데이터 처리 시스템의 부재로 발생하는 복잡도를 줄이기 위해 탄생한 Consumer와 Producer가 분리된 카프카(Kafka)의 특징을 배웁니다.
카프카와의 직접적인 통신을 통해 카프카의 메타 데이터 정보를 저장하고 상태를 관리하는 주키퍼(Zookeeper)에 대해 학습합니다.
이후, Docker와 Python을 활용한 카프카 실습을 통해 메시지와 토픽을 생성하고 전달해보는 등의 운영을 공부합니다.
